4 types:
1. public: Accessible from anywhere
2. private: Accessible only inside the class. Cannot access even after inheriting
3. protected: Accessible in class and derived classes
4. internal: Accessible within the same project/assembly

Default Access modifier:
- **Class**: internal
- **Methods and Fields**: private


**Accessors**
- get (read)
- set (write)
- init (write-only during initialization)
